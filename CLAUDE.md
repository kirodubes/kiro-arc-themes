# CLAUDE.md — kiro-arc-themes

## Project overview

See [README.md](./README.md) for the user-facing description. This is the
**data repo**: the built output of the full Arc theme collection — 55 colours ×
4 tones = 220 theme folders under `usr/share/themes/`.

## Current state

- Themes are **generated**, not hand-edited. Source of truth for the build is
  [kiro-arc-themes-generator](/home/erik/KIRO/kiro-arc-themes-generator)
  (`2-make-all-themes-for-arcolinux.sh`). To regenerate, rebuild into
  `OUTPUT_DIR=usr/share/themes` — do **not** edit theme files by hand.
- The single-colour **Arc Dawn** family is shipped from
  [kiro-arc-dawn](/home/erik/KIRO/kiro-arc-dawn) and is deliberately excluded
  here (the generator's `SKIP="dawn"` mirrors this).
- Packaged by the split PKGBUILD in
  `~/KIRO-PKG-BUILD-APPS/kiro-arc-themes/`, generated by the generator's
  `3-make-pkgbuild.sh`, which reads the colour list from `usr/share/themes/`.

## Patterns & decisions

- Layout must match `kiro-arc-dawn`: themes live under `usr/share/themes/Arc-*`,
  never at the repo root (the generator and PKGBUILD both assume this path).
- Package names: `kiro-arc-<colour>` (lowercase), one package per colour,
  `replaces=('arcolinux-arc-<colour>-git')`.
- gtk3/gtk4 accent comes from the meson `accent` option — `sed` alone leaves gtk
  blue. xfwm/unity/metacity/cinnamon are recoloured by `sed`.
- Bash scripts (`setup.sh`, `up.sh`) follow the canonical Kiro-HQ template — see
  [Kiro-HQ/CLAUDE.md](/home/erik/Insync/Kiro/Kiro-HQ/CLAUDE.md).
